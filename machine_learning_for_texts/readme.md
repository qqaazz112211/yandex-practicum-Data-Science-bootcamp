
| Название проекта | Описание |

**Используемые библиотки:**
- pandas
- numpy 
- matplotlib
- plotly.express
- scikit-learn
- seaborn
- statsmodels
- catboost
- os
- xgboost
- lightgbm
- Sktime
- ETNA
